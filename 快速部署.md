# 🚀 快速部署脚本

## 方法 1：使用 HuggingFace CLI（推荐）

### 安装 HuggingFace CLI
```bash
pip install huggingface_hub
```

### 登录 HuggingFace
```bash
huggingface-cli login
```
然后输入你的 HuggingFace token（可在 https://huggingface.co/settings/tokens 获取）

### 创建并上传 Space
```bash
# 创建 Space 目录
cd d:\神经网络\digit-recognizer

# 上传到 HuggingFace Spaces
huggingface-cli upload yuan-linyang/mnist-digit-recognition ./app.py ./app.py
huggingface-cli upload yuan-linyang/mnist-digit-recognition ./best_model.pth ./best_model.pth
huggingface-cli upload yuan-linyang/mnist-digit-recognition ./requirements.txt ./requirements.txt
huggingface-cli upload yuan-linyang/mnist-digit-recognition ./README.md ./README.md
huggingface-cli upload yuan-linyang/mnist-digit-recognition ./.gitignore ./.gitignore

echo "✅ 部署完成！访问：https://huggingface.co/spaces/yuan-linyang/mnist-digit-recognition"
```

## 方法 2：使用 Git

### 克隆 Space 仓库
```bash
git clone https://huggingface.co/spaces/yuan-linyang/mnist-digit-recognition
cd mnist-digit-recognition
```

### 复制文件
```bash
# Windows PowerShell
Copy-Item ..\digit-recognizer\app.py .
Copy-Item ..\digit-recognizer\best_model.pth .
Copy-Item ..\digit-recognizer\requirements.txt .
Copy-Item ..\digit-recognizer\README.md .
Copy-Item ..\digit-recognizer\.gitignore .

# 或者在文件管理器中手动复制
```

### 提交并推送
```bash
git add .
git commit -m "Deploy MNIST digit recognition app

Features:
- Image upload recognition
- Online sketchpad input
- Top-3 prediction with confidence
- Probability distribution visualization

Model:
- CNN with 99.35% validation accuracy
- Adam optimizer + Data Augmentation + Early Stopping

Author: Yuan Linyang (112304260141)"
git push
```

## 方法 3：网页上传（最简单）

### 步骤
1. 访问 https://huggingface.co/spaces/yuan-linyang/mnist-digit-recognition
2. 点击 "Files" → "Add file" → "Upload files"
3. 拖拽以下文件到上传区域：
   - app.py
   - best_model.pth
   - requirements.txt
   - README.md
   - .gitignore
4. 点击 "Commit changes to main"
5. 等待 2-5 分钟部署完成

## ⚠️ 常见问题

### Q1: 上传失败 "Repository not found"
**A**: 需要先创建 Space，然后再上传文件

### Q2: 部署失败 "Error"
**A**: 
1. 点击 "Logs" 查看错误信息
2. 检查 requirements.txt 是否完整
3. 确保模型文件路径正确

### Q3: 页面加载慢
**A**: 首次加载需要加载模型（5-10 秒），属正常现象
