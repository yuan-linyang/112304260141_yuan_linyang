## 🌐 部署步骤摘要

### GitHub 部署
```bash
cd digit-recognizer
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/yuan-linyang/112304260141_yuan_linyang.git
git push -u origin main
```

### Render 部署
1. 登录 https://dashboard.render.com/ (使用 GitHub 账号)
2. New + → Web Service
3. 选择仓库 `112304260141_yuan_linyang`
4. 配置:
   - Name: `mnist-digit-recognition`
   - Region: `Singapore`
   - Root Directory: `digit-recognizer`
   - Build Command: `pip install -r requirements.txt`
   - Start Command: `python app_final.py`
5. 点击 Create Web Service
