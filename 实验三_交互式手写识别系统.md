
## 3.6 Render 云平台部署

### 步骤 1: 登录 Render

1. 访问 https://dashboard.render.com/
2. 点击 "Sign in with GitHub"
3. 授权 Render 访问你的 GitHub 账号

### 步骤 2: 创建 Web Service

1. 点击 "New +" → "Web Service"
2. 选择你的仓库：`112304260141_yuan_linyang`
3. 配置如下：

**Basic Settings**:
- **Name**: `mnist-digit-recognition`
- **Region**: Singapore (新加坡，离中国最近)
- **Branch**: main
- **Root Directory**: `digit-recognizer`
- **Runtime**: Python 3
- **Build Command**: `pip install -r requirements.txt`
- **Start Command**: `python app_final.py`

**Environment Variables**:
```
PYTHON_VERSION=3.8.0
```

**Instance Type**:
- 选择 **Free** 套餐

### 步骤 3: 部署

1. 点击 "Create Web Service"
2. 等待构建和部署（约 3-5 分钟）
3. 部署成功后，复制生成的 URL（如：`https://mnist-digit-recognition.onrender.com`）

### 步骤 4: 配置 render.yaml（可选）

在项目根目录创建 `render.yaml`:

```yaml
services:
  - type: web
    name: mnist-digit-recognition
    env: python
    region: singapore
    plan: free
    buildCommand: pip install -r requirements.txt
    startCommand: python app_final.py
    envVars:
      - key: PYTHON_VERSION
        value: 3.8.0
    disk:
      name: model-data
      mountPath: /opt/render/project/src
      sizeGB: 1
```

## 3.12 可能的问题与解决方案

### 问题 1: Render 部署后模型加载失败
**解决方案**: 确保模型文件通过 Git LFS 上传，检查文件大小限制

### 问题 2: 识别准确率不高
**解决方案**: 
- 确保绘制的数字居中
- 使用黑色画笔，白色背景
- 数字大小适中

### 问题 3: 访问速度慢
**解决方案**: 
- Render 免费套餐有冷启动时间
- 首次访问需等待 30-60 秒
- 后续访问会更快
